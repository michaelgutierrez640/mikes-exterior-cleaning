## Tips

- **JPG or WebP** for photos — keep gallery images under 300 KB each for fast loading
- **MP4** for videos — 1080p max, under 10 MB
- File names must match exactly (e.g. `01.jpg`, not `01.JPG` unless you update the config)
- Each placeholder on the live site shows the exact path and recommended size for that slot
